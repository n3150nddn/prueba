# Prueba

Repositorio de prueba para probar la funcionalidad de los comandos de git

# Código de ejemplo
```go
package main

import (
    "fmt"
)

func main() {
    fmt.Println("Hola mundo")
}
```

### Sistemas Operativos

También puede compilar programas para ejecutarlos en:
- Linux 
- macOS 
- Windows
